typedef void(* httpRecv_func) (dataCntxt_t sckt, uint16_t httpRslt, char *data, uint16_t dataSz)

Callback function for data received event. Notifies application that new data is available and needs serviced.

The *data and dataSz values are for convenience, since the application supplied the buffer to LTEmC.

Parameters
dataCntxt[in] Data peer (data context or filesys)
httpRslt[in] HTTP status code found in the page result headers
data[in] Pointer to received data buffer
dataSz[in] The number of bytes available

◆ http_cancelPage()

void http_cancelPage ( httpCtrl_t *  httpCtrl)

Cancels a http_readPage flow if the remaining contents are not needed.

This is a blocking call. The page read off the network will continue, but the contents will be discarded.

Parameters
httpCtrl[in] - Pointer to the control block for HTTP communications.
