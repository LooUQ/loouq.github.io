◆ mqttRecv_func

typedef void(* mqttRecv_func) (dataCntxt_t dataCntxt, uint16_t msgId)

Callback function to signal the availability of a received MQTT message.

The host application will need to use mqtt_readTopic(), mqtt_readMessage() to capture the received information, or

use the mqtt_signalComplete() to tell driver to complete the receive and discard any un-read information (release resources).

Parameters
dataCntxt[in] The data context receiving data.
msgId[in] MQTT ID of the message received.
