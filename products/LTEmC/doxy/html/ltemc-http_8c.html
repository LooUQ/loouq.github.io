◆ http_cancelPage()

void http_cancelPage ( httpCtrl_t *  httpCtrl)

Cancels a http_readPage flow if the remaining contents are not needed.

This is a blocking call. The page read off the network will continue, but the contents will be discarded.

Parameters
httpCtrl[in] - Pointer to the control block for HTTP communications.
