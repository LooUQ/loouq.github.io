Configure the TLS/SSL settings for a context.

The TLS/SSL context is loosely associated with the protocol context. This package maintains a 1-to-1 map for consistency.

Parameters
contxt[in] TLS/SSL context to configure
version[in] TLS/SSL version: 0=SSL-3.0, 1=TLS-1.0, 2=TLS-1.1, 3=TLS-1.2, 4=ALL
cipherSuite[in] Cipher suite to use for processing of crypto
certExpirationCheck[in] Options for how the certificate's expiration information is processed
securityLevel[in] Authentication mode: 0=no auth, 1=server auth, 2=server/client auth
