void IOP_forceTx ( const char *  sendData,
uint16_t  sendSz 
)

Perform a forced TX send immediate operation. Intended for sending break type events to device.

sendData must be less than 64 chars. This function aborts any TX and immediately posts data to UART.

Parameters
sendData[in] Pointer to char data to send out, input buffer can be discarded following call.
sendSz[in] The number of characters to send.

Perform a TX send operation.

This call will block until TX is fully sent out to UART. The BGx module is synchronous, so no buffering is implemented (may change).

Parameters
