Detailed Description

template<typename T>
class iotQc::memory::SyncPointer< T >

A smartpointer that syncs new pointers to a specific location.

When a pointer is created, it defines a location to point to. This pointer directs all new derived pointers to this same location in order to sync the object they all point to.
