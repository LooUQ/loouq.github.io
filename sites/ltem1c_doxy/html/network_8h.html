◆ protocol_t

enum protocol_t

Enum of protocols available on the modem.

Note: all of the protocols are clients, while the BGx line of modules support server mode the network carriers generally don't

Enumerator
protocol_tcp 

TCP client.

protocol_udp 

UDP client.

protocol_ssl 

SSL client.

protocol_AnyIP 

special value that includes any of the above IP basic transport protocols.

protocol_http 

HTTP client.

protocol_https 

HTTPS client, HTTP over SSL.

protocol_mqtt 

MQTT messaging client.

protocol_mqtts 

MQTT over SSL.

protocol_void 

No protocol, used in code to generally signal a null condition.
