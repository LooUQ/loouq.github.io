◆ ltem1Functionality_t

enum ltem1Functionality_t

Enum representing the service functionality to turn-on during LTEm1 initialization.

Enumerator
ltem1Functionality_stop 

Default state, no services set to start.

ltem1Functionality_base 

Basic services only, GPIO, SPI Bridge, BGx.

ltem1Functionality_iop 

Start IOP (Input/Output Processor)

ltem1Functionality_actions 

Start actions services (AT commands)

ltem1Functionality_services 

Start application services (protocols: sockets[TCP\UDP\SSL], MQTT, HTTP, etc.)
