mainpage LooUQ Circuit River LTEm1 modem driver implemented in C99 for portability and a small footprint The LooUQ LTEm1c driver supports the LooUQ Circuit River LTEm1 LTE modem The LTEm1 is designed to allow for cost effective cellular communications in embedded systems by supporting LTE CAT M1 and LTE CAT NB1 (NB-IOT) communications. The LTEm1 is certified for use on the Verizon network and a registered device for the T-Mobile(formerly Sprint) Curiosity Network. It is also fully tested and supported using Hologram SIMs. *1st off
