Open a data connection (socket) to d data to an established endpoint via protocol used to open socket (TCP/UDP/TCP INCOMING).

Parameters
[in]socketId- The ID or number specifying the socket connect to open.
[in]protocol- The IP protocol to use for the connection (TCP/UDP/TCP LISTENER/UDP SERVICE/SSL).
[in]host- The IP address (string) or domain name of the remote host to communicate with.
[in]rmtPort- The port number at the remote host.
[in]lclPort- The port number on this side of the conversation, set to 0 to auto-assign.
[in]rcvr_funcThe callback function in your application to be notified of received data ready.
