void sckt_cancelRecv ( scktCtrl_t *  scktCtrl)

Cancel an active receive flow and discard any recieved bytes.

This is a blocking call, returns after all outstanding bytes are retrieved from module and discarded. Connection can continue receiving new bytes if not closed.

Parameters
scktCtrl[in] Pointer to socket control being operated on.
