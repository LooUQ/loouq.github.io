◆ atcmd_defaultResultParser()

resultCode_t atcmd_defaultResultParser ( const char *  response,
const char *  preamble,
bool  preambleReqd,
uint8_t  gapReqd,
const char *  terminator,
char **  endptr 
)

Performs a standardized parse of command responses. This function can be wrapped to match atcmd commandCompletedParse signature.

Parameters
response[in] - The string returned from the getResult function.
preamble[in] - The string to look for to signal start of response match.
preambleReqd[in] - The preamble string is required, set to false to search for only gap and terminator
gapReqd[in] - The minimum char count between preamble (or start) and terminator (0 is valid).
terminator[in] - The string to signal the end of the command response.
endptr[out] - Char pointer to the next char in response after parser match
Returns
HTTP style result code, 0 if parser incomplete (need more response). OK = 200.
