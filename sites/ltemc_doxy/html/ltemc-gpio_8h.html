resultCode_t gpio_configPort ( uint8_t  portNumber,
gpioDirection_t  direction,
gpioPull_t  pullType,
gpioPullDrive_t  pullDriveCurrent 
)

Configure a GPIO port for intended use.

Parameters
[in]portNumber- The GPIO port to configured, dependent on modem module and modem board.
[in]direction- Set the GPIO port to be for input or output.
[in]pullType- If the port is input, specifies the optional pull up/down behavior for the GPIO port. Ignored if "direction" is output.
[in]pullDriveCurrent- If the port is input AND pull is specified, sets the current limit for the pull circuit. Ignored if "direction" is output.
Returns
Value read 0=low, 1=high. Result of -1 indicates invalid port number.
