{\footnotesize\ttfamily typedef volatile struct \mbox{\hyperlink{structiop__tag}{iop\+\_\+tag}} \mbox{\hyperlink{ltemc-iop_8h_a85a71d0ff773154d7fb17d9a6589b3f0}{iop\+\_\+t}}}



Struct for the I\+OP subsystem state. During initialization a pointer to this structure is reference in g\+\_\+ltem1. 

Each of the protocols (sockets, M\+Q\+TT, H\+T\+TP) have unique behaviors\+: sockets receive asynchronous alerts but the data receive process is fully synchronous, H\+T\+TP is synchronous on receive tied to page read, M\+Q\+TT is fully asynchronous with msg event receive signaling and data transfer taking place async behind the scenes initiated on interrupt.
