Two callback conventions have be formalized\+: yield and notification. The yield callback allows for the application to invoke watchdog keepalives whenevr the L\+T\+Em1c code passes through a wait for response block. This is optional and is specified with the {\itshape ltem1\+\_\+set\+Yield\+Cb(yield\+Cb)} method. The yield\+Cb() method takes no parameters and returns a void.

The other callback is for notifying your application of significant events, such as\+: connecting, disconnecting and errors. This is compatible with the notification callback used in the L\+Q\+Cloud driver. An example callback handler is shown below. \hypertarget{md__release_1_0_autotoc_md27}{}\doxysubsection{New Setup and Start, Protocol Inclusion}\label{md__release_1_0_autotoc_md27}
In your application\textquotesingle{}s setup routine the functions to create/initialize the L\+T\+Em1 and to start the modem has been separated into distinct methods. The new approach is to create the required features, then issue a ltem1\+\_\+start() call.


\begin{DoxyCode}{0}
\DoxyCodeLine{ltem1\_create(ltem1\_pinConfig, appNotifyCB);}
\DoxyCodeLine{sckt\_create();}
\DoxyCodeLine{ltem1\_start(pdpProtocol\_sockets);}
\end{DoxyCode}


The application notification callback is optional and can be disabled with {\bfseries{ltem1\+\_\+create(ltem1\+\_\+pin\+Config, N\+U\+L\+L);}}

Optional Protocols are are bit flags that are binary or\textquotesingle{}d together. Available options are\+:
\begin{DoxyItemize}
\item pdp\+Protocol\+\_\+none // access to only network carrier status functionality
\item pdp\+Protocol\+\_\+sockets // T\+C\+P/\+U\+D\+P/\+S\+S\+L/\+T\+LS P\+O\+S\+IX style sockets
\item pdp\+Protocol\+\_\+mqtt // M\+Q\+TT message queueing
\end{DoxyItemize}

To use both sockets and M\+Q\+TT you would initialize the L\+T\+Em1 with... 
\begin{DoxyCode}{0}
\DoxyCodeLine{ltem1\_create(ltem1\_pinConfig, appNotifyCB);}
\DoxyCodeLine{sckt\_create();}
\DoxyCodeLine{mqtt\_create();}
\DoxyCodeLine{ltem1\_start(pdpProtocol\_sockets | pdpProtocol\_mqtt);}
\end{DoxyCode}
